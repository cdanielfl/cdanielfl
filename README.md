### [Soffta — Sistema de Gestão de Filas para UPAs](https://github.com/cdanielfl)
> Sistema web completo para orquestrar o fluxo de atendimento em clínicas e Unidades de Pronto Atendimento.

- Arquitetura **multi-tenant** com isolamento de dados por unidade de saúde
- Autenticação com **RBAC** — 4 hierarquias de acesso (Super Admin, Admin UPA, Recepcionista, Atendente)
- Painel público para TVs com **Text-to-Speech (TTS)** para chamada de pacientes
- Dashboards analíticos com métricas hospitalares em tempo real
- Suporte a **White Label** com temas Dark/Light Mode por unidade
- **Stack:** React.js · Vite · Tailwind CSS · Zustand · Framer Motion · Recharts · Node.js · Express.js · PostgreSQL

---

### [Plataforma SaaS — Empresas de Higienização](https://github.com/cdanielfl)

- Pipeline completo: desenvolvimento → testes (Jest) → deploy (Vercel/Render)
- **Stack:** React.js · Node.js · Express.js · PostgreSQL · Git · GitHub
